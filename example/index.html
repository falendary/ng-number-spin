Customizable directive for numeric inputs ( <input type="number"> ) for Angular.js (1.6.4)

Example 1

Default settings

HTML
<number-spin data-ng-model="vm.amount"></number-spin>
Javascript
app.controller('Example1Controller', ['$scope', function($scope){ var vm = this; vm.amount = 10; }]);

Example 2

Custom step

HTML
<number-spin data-ng-model="vm.amount" data-step="vm.step"></number-spin>
Javascript
app.controller('Example2Controller', ['$scope', function($scope){ var vm = this; vm.amount = 10; vm.step = 4; }]);

Example 3

Hide spin buttons

HTML
<number-spin data-ng-model="vm.amount" data-spin-buttons="vm.showButtons"></number-spin>
Javascript
app.controller('Example2Controller', ['$scope', function($scope){ var vm = this; vm.amount = 10; vm.vm.showButtons = false; }]);
